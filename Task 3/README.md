# Iris Flower Classification

## Project Overview
This project is developed as part of the **CodSoft Data Science Internship (Task 3)**.

The objective is to build a Machine Learning model that classifies Iris flowers into one of three species based on the flower's measurements.

The three species are:
- Iris-setosa
- Iris-versicolor
- Iris-virginica



## Dataset

The dataset contains **150 samples** with the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Target)



## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn



## Machine Learning Algorithm

- Random Forest Classifier



## Steps Performed

1. Imported required libraries
2. Loaded the dataset
3. Explored the dataset
4. Visualized the data
5. Split the dataset into training and testing sets
6. Trained the Random Forest model
7. Predicted flower species
8. Evaluated model performance
9. Predicted the species of a new flower


## Evaluation Metrics
