# Sales Prediction Using Python

## Project Overview

This project predicts product sales based on advertising expenditures across different media platforms such as TV, Radio, and Newspaper using Machine Learning in Python.

A Linear Regression model is trained on historical advertising data to estimate future sales based on advertising budgets.

---

## Objective

The objective of this project is to build a machine learning model that can accurately predict sales using advertising expenditure data.

---

## Dataset

The dataset contains the following features:

- TV – Advertising budget spent on TV
- Radio – Advertising budget spent on Radio
- Newspaper – Advertising budget spent on Newspaper
- Sales – Product sales (Target Variable)

---

## Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Exploration
4. Exploratory Data Analysis (EDA)
5. Data Preparation
6. Train-Test Split
7. Train Linear Regression Model
8. Predict Sales
9. Evaluate Model Performance
10. Predict Sales for New Data

---

## Model Evaluation

The model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
